# Research Software Readiness Checker Pro

## Features
- Analyzes README, License, Tests, CI/CD, Versioning, Citation
- Weighted scoring out of 100
- PDF, HTML, JSON export
- History tracking
- Confetti for high scores

## How to Use
1. Open index.html in Chrome/Edge
2. Click "Select Folder"
3. View results
